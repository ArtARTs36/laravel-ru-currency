This package provides currencies with courses For Laravel

## Installation

Run: `composer require artarts36/laravel-ru-currency`

## Commands for Application

| Command                   | Description          |
|---------------------------|----------------------|
| ru-currency:fetch-courses | Fetch actual courses |

## Development Console
